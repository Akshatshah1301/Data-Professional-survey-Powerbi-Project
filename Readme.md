# Data Professional Survey Dashboard (Power BI)

This is a Power BI dashboard built using survey data of data professionals.

## Files
- Power Bi project.pbix → Power BI dashboard
- Power BI - Final Project.xlsx → Dataset
- Dashboard.png → Dashboard screenshot

## Dashboard Shows
- Country of survey takers
- Average salary by job title
- Favorite programming language
- Work life balance score
- Salary happiness
- Difficulty to break into data roles

## How to Open
1. Download Power Bi project.pbix
2. Open using Power BI Desktop
